Концерт В. Джиоевой

1.1. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой,
2 шт., на стойке с круглым основанием используется
для персонального озвучивания солиста. (КМ24К1, КМ24К2)

1.2. Микрофон граничного слоя устанавливаются на край аванс-сцены
для озвучивания группы солистов. (КМ24К3)

1.3. Микрофон граничного слоя устанавливаются на край аванс-сцены
для озвучивания группы солистов. (КМ23К1)

1.4. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 2 шт.,
на стойке с круглым основанием используется для персонального озвучивания солиста. (КМ23К2, КМ23К3)

1.5. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 2 шт. на стойке
используется для озвучивания первого и второго пультов группы I скрипок. (КМ19К1, КМ19К2)

1.6. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 2 шт. на стойке
используется для озвучивания первого и второго пультов группы II скрипок. (КМ19К3, КМ19К4)

1.7. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 2 шт. на стойке
используется для озвучивания первого и второго пультов группы альтов. (КМ20К1, КМ20К2)

1.8. Микрофон конденсаторный, кардиоидный, с широкой диафрагмой, 2 шт. на низкой стойке
используется для озвучивания первого и второго пультов группы виолончелей. (КМ20К3, КМ20К4)

1.9. Микрофон конденсаторный, кардиоидный, с широкой диафрагмой, 2 шт. на низкой стойке
используется для озвучивания первого и второго пультов группы контрабасов. (КМ21К1, КМ21К2)

1.10. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 5 шт. на стойке
используется для озвучивания группы деревянных духовых инструментов. (КМ13К1, КМ13К2, КМ13К3, КМ13К4, КМ13К5)

1.11. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 1 шт. на стойке
используется для озвучивания арфы. (КМ13К6)

1.12. Микрофон конденсаторный, кардиоидный, с узкой диафрагмой, 5 шт. на стойке
используется для озвучивания группы медных духовых инструментов. (КМ14К1, КМ14К2, КМ14К3, КМ14К4, КМ14К5)

1.13. Микрофон конденсаторный, кардиоидный, с широкой диафрагмой, 2 шт. на стойке
используется для озвучивания литавр и ударно-перкуссионной группы. (КМ14К6, КМ14К7)

1.14. Стереопара (XY), 3шт., состоящая из конденсаторных кардиоидных микрофонов,
устанавливается перед хором. (КМ15К1, КМ15К2, КМ15К3, КМ15К4, КМ12К1, КМ12К2)
